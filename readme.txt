This is a simple Python app built using Tkinter. It has a main page with two buttons. One page takes you to the skincare quiz and the other button takes you to a catalog of product recommendations. The quiz helps users figure out their skin type (dry, oily, or normal) based on their answers.

Our app features
A skincare quiz with multiple-choice questions
Real time results
A product catalog page
